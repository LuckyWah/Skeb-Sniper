Release Log

Version 1.0 (12/27/2024)

This initial release includes a basic Python script with the core functionality of automation for Windows servers and provided a tutorial for first-time users.

Version 1.1 (12/28/2024)

This update provided a dedicated version for Linux servers along with extensive guidance for setting up cloud servers.

Version 1.2 (12/29/2024)

Anti-detection measures were introduced to avoid bot detection. This update implements a randomized User-Agent with fallback mechanism, disables headless mode, activates privacy protection such as tracking protection, and randomizes page check intervals between 5 seconds and 10 minutes.

Version 1.3 (12/30/2024)

Created Windows and Linux setup scripts, simplifying the application’s installation for non-technical users.

Version 1.4 (12/31/2024)

Added encryption mechanism to ensure security of Twitter credentials.

Version 1.5 (01/01/2025)

A graphical user interface (GUI) was implemented with data encryption for usability and security. Users can now safely enter their Twitter credentials and commission details via the GUI.

Version 1.6 (01/02/2025)

The Linux version became fully operational with data encryption, using the command prompt for user input instead of a GUI. Enhancements include an improved manual, bug fixes, logging, and a locking mechanism to prevent duplicate requests.

Version 1.7 (01/03/2025)

Bug fixes for selecting toggles. The users can now create multiple artist-specific Python files named after each artist to monitor commission statuses simultaneously.

Version 1.8 (01/29/2025)

Significant enhancements were made to the Windows GUI, enabling users to manage tasks entirely through the graphical interface and integrating relevant information to reduce the need for tutorials.

Version 1.9 (03/15/2025)

The applcation now operates as a fully event-driven tool. The application sends a single request to Skeb when an artist opens requests, which is triggered by Skeb’s email notifications thus eliminating the need for unethical web scraping. Windows and Linux versions are now available after purchase.
